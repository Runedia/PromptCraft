# src/core — 핵심 비즈니스 로직 레이어

**목적:** 스캔, Q&A, 프롬프트 빌드 등 핵심 기능의 순수 로직. 어떤 인터페이스에도 의존하지 않음.

## 컨벤션

- **LLM API 호출 완전 금지** — 외부 AI 서비스 의존 금지
- **Commander.js / Express import 금지** — 프레임워크 종속 없는 순수 TS
- SQLite 접근 시 `~/.promptcraft/` 경로만 사용
- 성능 기준: 스캔 5초 이내, 프롬프트 빌드 5초 이내

## 패턴

- 모든 I/O는 인터페이스 레이어(cli/server)에서 처리
- 반환값은 직렬화 가능한 순수 객체
