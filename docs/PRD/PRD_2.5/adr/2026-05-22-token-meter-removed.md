# 2026-05-22 — §3.3.3 (토큰 사용량 계측 인프라) 삭제

**결정**: 토큰 사용량 자체 측정 인프라(`usage.sqlite` 테이블 등)를 PRD에서 제거.

**근거**:
1. **외부 도구 위임**: `getagentseal/codeburn` 같은 토큰 추적 전용 도구가 이미 존재한다. PromptCraft가 중복 구현할 이유 없음.
2. **본질적 검증 불가능성**: LLM 응답은 비결정적이라 동일 입력·동일 모델에서도 토큰 사용량 분산 큼. R1 L2-A 재시도(`test003_2`)에서 S → A로 등급조차 회귀한 사실이 직접 입증. 토큰 효율의 A vs B 비교는 다회 측정으로도 통계적 유의성 확보 어려움.
3. **인과 추론 불가**: "PromptCraft가 토큰을 절약했다"가 아니라 "응답이 짧았다"만 측정되며, 짧은 응답이 좋은 응답인지도 불명.

**연쇄 정리**:
- KPI #3 (총 토큰 소비량 감소율) 삭제. KPI 18개 → 16개 재번호화.
- §7.2.1 P0 4건 → 1건. 토큰·latency·메타데이터 스펙 3건 삭제, 모델 식별 메타데이터만 P0-1로 유지.
- §7.3 PRD 2.6 예상 영역에서 "토큰 효율 가설 재검증" 삭제.
- H3/H6 "검증 불가" → "검증 보류 (외부 도구 위임)"로 표현 변경.
