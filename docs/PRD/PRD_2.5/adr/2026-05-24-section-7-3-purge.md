# 2026-05-24 — §7.3 "PRD 2.6 예상 영역" 절 통째 폐기

**결정**: §7.3 "PRD 2.6 예상 영역" 섹션(헤딩 + 설명 문단 + 예고 표 1행)을 통째 삭제한다. §7.4 "평가 가이드라인"은 **재번호하지 않고 §7.4로 유지**한다. §7은 §7.1·§7.2·§7.4로 마무리되며 §7.3·§7.5는 결번 상태로 남는다([[2026-05-23-section-7-5-purge]] 선례 계승).

**근거**:
1. **자기 규정상 비공식 예고**: §7.3 본문은 "본 절은 비공식 예고이며 PRD 2.6 작업 시 재검토합니다"로 스스로를 규정했다. 실측·결정 기반이 아닌 추정성 예고로, PRD living spec의 "본문은 확정 spec만" 원칙과 부합도가 낮다.
2. **잔존 1행 + 의미론적 약화**: 본래 §7.3은 여러 예고 행("토큰 효율 가설 재검증" [[2026-05-22-token-meter-removed]], "모델별 강점 패턴 카탈로그" [[2026-05-23-p2-purge-8]], "응답-구현 정합성 자동 검증 결과" [[2026-05-24-kpi14-cancel]])을 담았으나 모두 선행 ADR로 삭제되어 "신규 카드 풀 추가" 1행만 남았다. 게다가 그 1행의 의존이던 P1-2는 [[2026-05-24-p1-2-cancel]]로 취소, P1-3은 [[2026-05-24-p1-3-redefine]]로 "환경 제약 컨텍스트 보강"으로 재정의되어 "신규 카드 풀 추가"와의 의미론적 연결이 약화됐다.
3. **보존 실익 없음**: PRD 2.6 착수 시 예상 영역은 어차피 신규 작성·재검토된다. 예고를 지금 유지할 실익이 없고, 삭제해도 정보 손실이 없다(P1-3·P1-4 과제 자체는 [[7.Roadmap|§7.2.2]]에 존재).
4. **사용자 판단**: 해당 절은 작성 과정에서 자동 생성된 예고이며 사용자 의도가 아니라는 확인. 불요 판단.
