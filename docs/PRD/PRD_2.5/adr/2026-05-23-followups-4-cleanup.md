# 2026-05-23 — 후속 과제 4건 정리 (완료 1건, 계획 취소 3건)

PRD §3/§4/§7에 누적된 후속 과제 표를 정합화. 본문은 living spec만 유지하고 취소·완료 항목은 본 문서에 기록한다.

## 1. 폰트 셀프 호스팅 — 완료

**결정**: Google Fonts CDN `@import` → `@fontsource/*` 셀프 호스팅 전환 완료([[5.Architecture|§5.4.1]]).

**근거**: [[5.Architecture|§5.2.1]] 네트워크 격리 원칙. Vite가 woff2/woff 파일을 자동 번들링하여 외부 CDN 호출 0건.

**연쇄 정리**:
- [[4.UI-Design|§4.1.3]] 헤딩/본문 갱신, Google Fonts CDN `@import` 코드 블록 제거
- [[4.UI-Design|§4.6]] 후속 개선 표에서 항목 1 (폰트 셀프 호스팅) 제거
- [[7.Roadmap|§7.2.2]] P1-5 삭제, P1-12 Settings UI 설명에서 "폰트 셀프 호스팅 토글" 제거

## 2. Tab / ⇧Tab 카드 포커스 이동 — 계획 취소

**결정**: `useKeyboard`의 Tab / ⇧Tab 인터셉트 구현 취소.

**근거**: 구현 가치 없음으로 판단. 브라우저 기본 Tab 동작 + 카드 내부 `focus-visible:ring`으로 baseline 접근성 충족.

**연쇄 정리**:
- [[3.Features|§3.2.11]] 표·결정 본문에서 항목 제거
- [[4.UI-Design|§4.3.5]] 키보드 표에서 행 제거, [[4.UI-Design|§4.6]] 후속 개선 표에서 항목 4 제거
- [[7.Roadmap|§7.2.2]] P1-13 삭제 (P1 번호는 결번 허용)

## 3. ⌘K 커맨드 팔레트 — 계획 취소

**결정**: 파워 유저 모드 ⌘K 커맨드 팔레트 도입 취소.

**근거**: 프로젝트 범위 초과. 사용 빈도 검토 결과 도입 가치 낮음으로 판단.

**연쇄 정리**:
- [[3.Features|§3.2.11]] 표·결정 본문에서 항목 제거
- [[4.UI-Design|§4.6]] 후속 개선 표에서 항목 12 제거
- [[7.Roadmap|§7.2.2]] P1-10 삭제

## 4. Dual-Pane 워크스페이스 옵션 — 계획 취소

**결정**: Three-Column 단일 레이아웃 확정. Dual-Pane 등 대안 사용자 선택형 옵션 도입 안 함.

**근거**: 현 Three-Column 레이아웃이 커밋 `c705d25`로 안정 정착. 대안 옵션 도입은 디자인 복잡도·Settings 노출·default 결정 등 부담 대비 가치 불명확.

**연쇄 정리**:
- [[3.Features|§3.2.4]] 본문에서 "대안 레이아웃 후속 검토" 문장 제거
- [[4.UI-Design|§4.6]] 후속 개선 표에서 항목 13 제거
- [[5.Architecture|§5.2.7]] 반응형 표에서 1024-1279px Dual-Pane 행 제거. ≥1024px 전체를 Three-Column으로 통합
- [[7.Roadmap|§7.2.2]] P1-9 삭제, [[7.Roadmap|§7.5]] "결정 필요" 표에서 P1-9 default 선택 행 제거
