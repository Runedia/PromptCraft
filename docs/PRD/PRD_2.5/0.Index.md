## 0. 본 문서의 위치

본 PRD는 7개 파트 + 본 Index로 분할되어 있습니다.

- 각 파트는 자기완결형 문서로 단독 독해가 가능합니다.
- 파트 간 참조는 상대 경로 링크(`./1.Overview.md` 등)로 명시합니다.
- 본 Index는 진입점 + 파트별 한 줄 요약 + 독해 가이드를 제공합니다.

## 2. 독해 가이드

### 2.1 처음 읽는다면

권장 순서: §1 Overview → §2 Users → §6 Evaluation → §3 Features → §4 UI-Design → §5 Architecture → §7 Roadmap.

- §1·§2가 제품 비전과 사용자 관점,
- §6이 본 버전의 핵심 동력(평가 결과),
- §3·§4·§5가 기능·디자인·기술 구현 명세,
- §7이 향후 작업 우선순위.

### 2.2 디자인 작업자라면

- §4 UI-Design 단독 독해 가능.
- §4.7 거버넌스 + §7.2.2 P1 디자인 시스템 과제만 참고하면 작업 범위 파악 가능.

### 2.3 평가 작업자라면

- §6 Evaluation + §7.4 평가 가이드라인 참조.
- 원본 평가보고서: [`시나리오/평가보고서.md`](../../evaluations/README.md), [`시나리오2/평가보고서.md`](../../evaluations/README.md).
