졸업작품 프로젝트 'PromptCraft'의 팀원 모집을 위한 홍보 이미지 제작을 요청드립니다.

[프로젝트 개요]
- 프로젝트명: PromptCraft (프롬프트크래프트)
- 한줄 슬로건: 로컬 코드 분석 기반 AI 코딩 프롬프트 & 컨텍스트 빌더
- 핵심 가치: "어떻게 질문할지 고민하지 마세요. PromptCraft가 프로젝트를 분석하고 질문을 완성해드립니다."

[이미지에 포함될 주요 키워드/내용]
1. 프로젝트 특징:
   - 프로젝트 자동 스캔 (언어, 프레임워크, 구조 분석)
   - 상황별 맞춤형 Q&A (에러 해결, 기능 구현, 코드 리뷰 등)
   - AI 컨텍스트 파일 생성 도움 (CLAUDE.md, GEMINI.md)
   - 로컬 우선 (Local-first), 오프라인 동작
2. 모집 대상: Node.js, React, CLI 도구 개발에 관심 있는 열정적인 개발자 (최대 4인 팀 구성 예상중)
3. 기술 스택: Node.js, React, SQLite, Express
4. 기간: 3~4개월 (졸업과제 프로젝트)

[비주얼 가이드 (협의 필요)]
- 전체적인 분위기: 개발자 감성이 느껴지는 모던한 테크/터미널 스타일
- 색상: 다크 모드 배경, 네온 블루 또는 그린 포인트 (터미널 프롬프트 느낌)
