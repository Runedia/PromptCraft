3. кі„мёөлі„ мұ…мһ„

3.1 Core Logic (src/core/)

лӘЁл“Ҳмұ…мһ„мһ…л Ҙм¶ңл Ҙ
Scannerл””л үнҶ лҰ¬ л¶„м„қкІҪлЎң(string)ScanResult к°қмІҙ
QnA Engineл¶„кё° нҠёлҰ¬ мҲңнҡҢ, лӢӨмқҢ м§Ҳл¬ё кІ°м •нҠёлҰ¬ID + нҳ„мһ¬ мқ‘лӢөлӢӨмқҢ м§Ҳл¬ё лҳҗлҠ” мҷ„лЈҢ мғҒнғң
Prompt BuilderмҲҳм§‘ лҚ°мқҙн„° вҶ’ н”„лЎ¬н”„нҠё мЎ°лҰҪScanResult + QnA мқ‘лӢөмҷ„м„ұлҗң н”„лЎ¬н”„нҠё(string)
Context Generatorм»Ён…ҚмҠӨнҠё нҢҢмқј мғқм„ұScanResult + мӮ¬мҡ©мһҗ м„Өм •нҢҢмқј лӮҙмҡ©(string)
