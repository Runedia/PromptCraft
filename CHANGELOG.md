# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1.0] - 2026-04-25

**PromptCraft 첫 공개 문서 릴리스: PRD 2.4, SWA 평가, Vibe 레벨 레퍼런스 추가.**

PRD 2.4 전체 사양, 2026년 4월 SWA(Software Architecture) 평가 결과, 그리고 AutoPlan 기능의 Vibe 레벨 레퍼런스 문서를 공개했습니다. 이 릴리스는 PromptCraft v0.0.1.0의 첫 번째 문서 체계 확립입니다.

### 이 버전의 핵심 수치

문서 체계 첫 릴리스로, 코드 측정 지표 대신 문서 커버리지로 평가합니다.

| 항목 | 내용 |
|------|------|
| PRD 2.4 사양 | 1,023줄 완전 사양 |
| SWA 평가 | 310줄 아키텍처 평가 |
| Vibe 레벨 레퍼런스 | 114줄 레퍼런스 |
| 총 추가 문서 | 1,447줄 |
